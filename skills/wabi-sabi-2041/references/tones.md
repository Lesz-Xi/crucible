# Tone Map

Select one tone per screen before composing visuals.

- `Quiet Precision`: clean, sparse, analytical, reassuring
- `Warm Intelligence`: humane, advisory, relational, soft authority
- `Aged Futurism`: timeworn materials paired with advanced capability
- `Ritual Flow`: deliberate pacing, reflective interactions, low-noise UX

If a screen mixes tones, anchor on one primary tone and use one secondary tone only for accents.
