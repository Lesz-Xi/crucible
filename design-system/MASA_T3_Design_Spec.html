Mode Group Labels

10px, JetBrains Mono, 600, --text-muted, uppercase, letter-spacing: 1px. Padding: 10px 12px 4px. Not clickable — structural only. Groups: Hybrid · Causal Chat · Legal.

Active state

Background: --bg-active (#EFF6FF). Text color: --accent. Font weight: 500. No left border stripe (too heavy). Matching T3 Code's subtle highlight approach.

Project row

Chevron ▾/▶ for collapse/expand. Font: 12px Inter 500 --text-primary. Clicking the chevron only — not the whole row — toggles collapse. Project label always visible.

New session button

Pinned to bottom of each mode group or sidebar footer. "+ New session" in --text-muted, 12px. On hover: --accent. No border, no background — ghost text link style.

Evidence Rail

Left border: 2px solid --border-focus. Background: --accent-subtle. Shown below AI responses whenever grounding is active. Each source is a clickable link opening the source document/URL. Label: GROUNDING SOURCES in 10px mono uppercase.

Confidence Bar

3px tall track (--border bg, --badge-novel fill). Width: 60px. Shown inline below the evidence rail. Followed by the numeric value in mono. Always visible — not collapsed behind a tooltip. Colors: green ≥0.8, yellow 0.5–0.79, red <0.5.

OODA Stage Bar (Hybrid only)

Horizontal inline progress in the mode bar. Stages: Observe · Orient · Decide · Act · Release. Completed stages show ✓ in --badge-novel. Active stage in --accent. Pending in --text-muted. Monospace 11px.

AI Avatar

22px circle. Background: --accent-light. Border: 1px solid --border-focus. Text: M (for MASA), 10px, --accent, 700 weight. Sits flush-top with the first line of the response. No shadow.

Borders & Elevation

No drop shadows on interactive components. Borders only. Elevation is expressed through background color difference, not shadow. Matching T3 Code's flat, zero-shadow aesthetic. The only permitted shadow is a focus ring: 0 0 0 3px --accent-light on inputs.

  • 2
