# MASA Design Tokens — CSS Variable Quick Reference

All variables are generated by Style Dictionary from `design-system/tokens.json`.
Output file: `synthesis-engine/src/app/generated-tokens.css`

---

## Core (`:root`) — Theme-agnostic

### Layout Dimensions
```css
--sidebar-w: 224px
--rail-w:    296px
--topbar-h:  42px
```

### Fonts
```css
--font-sans:  'Inter', -apple-system, sans-serif
--font-mono:  'JetBrains Mono', monospace
--font-serif: 'Instrument Serif', Georgia, serif
  aliases: --font-instrument-serif
```

### Border Radius
```css
--radius-sm:  5px
--radius:     8px
--radius-lg:  14px   (also 12px in prototype — use 12px for cards in Next.js)
--radius-xl:  18px
/* pill shapes: border-radius: 999px */
```

### Spacing Scale (px)
```
sp-1: 4   sp-2: 6   sp-3: 8   sp-4: 10  sp-5: 12
sp-6: 14  sp-7: 16  sp-8: 18  sp-9: 20  sp-10: 22
sp-12: 24 sp-14: 28 sp-16: 32 sp-20: 40 sp-24: 48
```

### Easings (not in tokens.json, hardcoded in prototype)
```css
--ease:        cubic-bezier(0.16, 1, 0.3, 1)
--ease-out:    cubic-bezier(0.0, 0.0, 0.2, 1)
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1)
```

## Dark Theme (`.dark`)

### Surfaces
```css
--bg:             #0E0D0C   /* main shell background */
--bg-2:           #131210   /* card-bg, input-bg */
--bg-3:           #1A1816   /* panel, field */
--bg-4:           #211F1C   /* soft panel, overlay */
--bg-5:           #282523   /* deep overlay */
--sidebar-bg:     #0B0A09   /* lab sidebar */
--rail-bg:        #0F0E0D   /* evidence rail */
--card-bg:        #131210   /* alias for --bg-2 */
--input-bg:       #1A1816   /* alias for --bg-3 */

/* Alt CSS aliases (for synthesis-engine components) */
--lab-bg:         #0E0D0C
--lab-bg-elevated: #131210
--lab-panel:      #1A1816
--lab-panel-soft: #211F1C
--lab-sidebar:    #0B0A09
--lab-shell-main: #0E0D0C

```

### Interactive States
```css
--bg-hover:   rgba(255,255,255,0.04)   /* subtle on hover */
--bg-active:  rgba(200,150,90,0.10)    /* amber-tinted for selected */
--lab-hover-bg:  rgba(255,255,255,0.04)
--lab-active-bg: rgba(200,150,90,0.10)
```

### Borders
```css
--border:    rgba(255,255,255,0.07)   /* default — very subtle */
--border-2:  rgba(255,255,255,0.12)   /* hover/focus elevation */
--border-3:  rgba(255,255,255,0.18)   /* strong, e.g. modals */
/* alias */ --codex-border: rgba(255,255,255,0.07)
/* alias */ --lab-border:   rgba(255,255,255,0.07)
```

### Text (warm white, opacity-ladder)
```css
--text-1:  #F2EFE9                    /* primary — warm off-white */
--text-2:  rgba(242,239,233,0.55)     /* secondary */
--text-3:  rgba(242,239,233,0.30)     /* tertiary / placeholders */
--text-4:  rgba(242,239,233,0.16)     /* quaternary / ghosts */
/* aliases */
--lab-text-primary:   #F2EFE9
--lab-text-secondary: rgba(242,239,233,0.55)
```

### Accent (amber)
```css
--accent:          #C8965A            /* primary amber */

--accent-dim:      rgba(200,150,90,0.10)   /* filled tint */

--accent-border-2: rgba(200,150,90,0.18)   /* softer border */

/* aliases */
--lab-accent-clay:  #C8965A
--lab-accent-rust:  #C8965A
--lab-accent-earth: #C8965A
```
